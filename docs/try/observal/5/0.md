`IfSetVal`|コールバック関数で真を返した時のみ代入する（それ以外は無視）
`ValidVal`|コールバック関数で真を返した時のみ`v`に代入し、それ以外は`iv`に代入する
`ChangedVal`|前回代入値と異なる値のとき`changed`コールバック関数を呼ぶ

`FixVal`|`IfSetVal`を継承したクラス。コールバック関数を渡さないと変更できず実質固定値になる。
